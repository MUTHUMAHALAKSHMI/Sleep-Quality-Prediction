# Sleep Quality Prediction using Machine Learning

This project predicts the sleep quality of a person based on:
- Sleep Duration
- Stress Level
- Physical Activity Level

## Algorithm Used
- Logistic Regression (Multiclass Classification)

## Features
- Predicts Sleep Quality (Poor / Average / Good)
- Provides health tips based on stress and activity
- Visualizes data using Matplotlib

## Technologies
- Python
- Pandas
- Scikit-learn
- Matplotlib

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run the project:
   python sleep_predictor.py

## Output
- Displays predicted sleep quality
- Shows data visualization graphs
